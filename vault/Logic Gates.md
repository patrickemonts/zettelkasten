A **logic gate** is a primitive that performs a Boolean function $f(x):\{0,1\}^n\rightarrow \{0,1\}^m$ .

Commonly used logic gates are 
- $AND$

| $AND$ | $0$ | $1$ |  
| -------- | -------- | -------- |  
| $0$ | $0$ | $0$ |  
| $1$ | $0$ | $1$ |

- $OR$

| $OR$ | $0$ | $1$ |  
| -------- | -------- | -------- |  
| $0$ | $0$ | $1$ |  
| $1$ | $1$ | $1$ |

- $NOT$

| $NOT$ | $0$ | $1$ |  
| -------- | -------- | -------- |  
|  | $1$ | $0$ |  


- $NAND$
	- The $NAND$ gate is universal, i.e. all other gates can be decomposed into $NAND$s.

| $NAND$ | $0$ | $1$ |  
| -------- | -------- | -------- |  
| $0$ | $1$ | $1$ |  
| $1$ | $1$ | $0$ |
